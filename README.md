# Experiment 40: Sorting Strings

## Problem Statement

Write a program to sort $N$ names in alphabetical order.

## Input Format

* First line: An integer $N$.
* Next $N$ lines: The names (strings).

## Output Format

* Print the sorted names, one per line.

### Example 1

**Input:**

```text
3
John
Alice
Bob
```

**Output:**

```text
Alice
Bob
John
```
